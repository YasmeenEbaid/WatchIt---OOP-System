# WatchIt-OOP-System
WatchIt is a Java-based Object-Oriented Programming project developed with JavaFX. The system demonstrates core OOP principles while providing a graphical user interface for better interaction and usability.
---
## 🚀 Features
- 🧱 Strong implementation of OOP principles  
- 🖥️ Graphical User Interface using JavaFX  
- 🎯 Modular and well-structured system design  
- ⚡ Interactive and user-friendly experience  
- 🔄 Organized classes and logic separation  
---
## 🛠️ Technologies Used
- Java  
- JavaFX  
- OOP Principles
